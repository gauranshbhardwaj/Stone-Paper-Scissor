A simple command-line Stone Paper Scissor game built using Python and NumPy. The player competes against the computer for a specified number of rounds. The computer randomly selects its move, and the program determines the winner of each round while keeping track of both player and computer scores.

Features
User-friendly command-line interface
Random computer choices using NumPy
Multiple rounds support
Automatic score tracking
Displays the winner after each round
Technologies Used
Python
NumPy

This project is ideal for beginners who want to understand Python functions, conditional statements, loops, random number generation, and the use of global variables.
